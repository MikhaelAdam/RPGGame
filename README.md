# Description
Proyek game ke-4 yang kita bikin. Gatau bakal selesai ato ngak

# Requirements
- CMake v3.30 
- git
- Mouse
- Keyboard
- Hands

# How to build
```
git clone https://github.com/MikhaelAdam/RPGGame.git
cd RPGGame
cmake -B build
cmake --build build
```
Or, use the build scripts
## Windows
```
.\build.bat
```
By default it's build for release. If you want to debug
```
.\build.bat Debug
```
## Linux
```
./build.sh
```
By default it's build for release. If you want to debug
```
./build.sh Debug
```
